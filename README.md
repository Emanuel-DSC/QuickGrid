## About this Project
Introducing our grocery shopping assistant app! This handy application aids you during your supermarket trips. It creates categorized item lists for easy retrieval and allows you to check off purchased items, saving the list for the next month to avoid re-adding everything. Developed with the purpose of studying state management in Flutter using Provider, it offers a seamless grocery shopping experience. Happy shopping! ðŸ›’ðŸ“±

## Why ? 
As I progressed in my studies, I felt the need to delve into state management, and I chose Provider as my go-to solution. Additionally, I was looking for a way to optimize my time during end-of-month grocery shopping, so I decided to combine the useful with the enjoyable.
