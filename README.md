## About this Project
Introducing our shopping assistant app! This handy application helps you during your shopping journeys. It creates categorized item lists for easy item retrieval and allows you to check off items you've already purchased, saving the list for the next month to avoid re-adding everything. Developed with the purpose of studying state management in Flutter using Provider, it offers a seamless shopping experience. Happy shopping! ðŸ›’ðŸ“±
